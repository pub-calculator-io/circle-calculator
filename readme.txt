Circle calculator finds missing characteristics of a circle. It includes a radius calculator, circumference calculator, diameter calculator, and circle area calculator.

[https://www.calculator.io/circle-calculator/](https://www.calculator.io/circle-calculator/)

== Usage ==

Add the Circle Calculator shortcode to your page, post or sidebar:

`[ci_circle_calculator]`

Add the following code to your template where you would like the Circle Calculator to appear:

`<?php display_ci_circle_calculator(); ?>`

== Installation ==

1. Upload the Circle Calculator /ci_circle_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Circle Calculator plugin through the "Plugins" menu in WordPress.
