== Description ==

Circle calculator finds missing characteristics of a circle. It includes a radius calculator, circumference calculator, diameter calculator, and circle area calculator.

Libraries in use:
1. https://mathjs.org/
2. https://katex.org/
3. https://github.com/aFarkas/lazysizes
4. https://github.com/RobinHerbots/Inputmask
5. https://air-datepicker.com/
6. https://www.chartjs.org/

== Installation ==

1. Upload the /ci_circle_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the [Circle Calculator](https://www.calculator.io/circle-calculator/ "Circle Calculator Homepage") plugin through the "Plugins" menu in WordPress.
3. Configure your settings via Appearance > Widgets > Circle Calculator.
4. And you're good to go!
